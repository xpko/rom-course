## 书籍简介

这是一部面向安卓软件开发与安全研究人员的技术指南，涵盖 Android 系统定制的核心知识。从基础环境搭建到深入修改 AOSP（Android 开源项目），本书提供清晰的实战案例，帮助读者快速掌握 Android 底层开发与定制的关键技术。

## 适用人群

- 需要定制 Android 设备的工程师
